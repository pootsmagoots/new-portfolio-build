Here is how I got started!

Hello there, welcome to my portfolio page. My story is rather simple. Have you ever had that feeling with a job that makes you feel a little empty inside? Like there is something missing? I had that feeling, in fact I had that feeling for quite some time.

However, along came computer programming and that forever has changed my life. Nothing has quite changed me so much (outside of my beautiful wife) than computer programming.

I have spent countless hours reading documentation, coding along with classes; once you experience seeing something that you have created, there is nothing quite like it. This has grown into my passion, and while there are challenges, the gratification that goes along with coding is truly something special. So with that, thanks for stopping by and hopefully I might become the developer you're looking for..

My Skills

Javascript

JavaScript, often abbreviated as JS, is a high-level, dynamic, weakly typed, prototype-based, multi-paradigm, and interpreted programming language.

CSS3

CSS3 is the latest evolution of the Cascading Style Sheets language and aims at extending CSS2.1. It brings a lot of long-awaited novelties, like rounded corners, shadows, gradients, transitions or animations, as well as new layouts like multi-columns, flexible box or grid layouts.

HTML5

HTML5 is a markup language used for structuring and presenting content on the World Wide Web. It is the fifth and current major version of the HTML standard.

Github

GitHub is a Web-based Git version control repository hosting service. It is mostly used for computer code. It offers all of the distributed version control and source code management functionality of Git as well as adding its own features.

GULP

gulp.js is an open-source JavaScript toolkit by Fractal Innovations and the open source community at GitHub, used as a streaming build system in front-end web development.

PostCSS

PostCSS is a software development tool that uses JavaScript-based plugins to automate routine CSS operations. The tool has been used to develop the code of Wikipedia, Facebook, and GitHub. PostCSS is a top-favored CSS tool among npm users.

WordPress

WordPress is a free and open-source content management system based on PHP and MySQL. To function, WordPress has to be installed on a web server, which would either be part of an Internet hosting service or a network host in its own right.

Node.JS

Node.js is an open-source, cross-platform JavaScript run-time environment for executing JavaScript code server-side.

Sass

Sass is a style sheet language initially designed by Hampton Catlin and developed by Natalie Weizenbaum.

BootStrap

Bootstrap is a free and open-source front-end web framework for designing websites and web applications.
